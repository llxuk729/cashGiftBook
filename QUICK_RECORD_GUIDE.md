# 快速记账功能说明

## 🚀 功能概述

快速记账是为了简化礼金记录流程而设计的功能，让用户能够在 **3秒内** 完成一笔记录，无需填写复杂的表单。

## ✨ 核心特性

### 1. **极简输入**
- ✅ 只需输入金额即可保存
- ✅ 姓名、备注都是可选的
- ✅ 自动使用默认分类
- ✅ 自动使用当前日期

### 2. **快捷金额按钮**
提供常用金额一键添加：
- +100
- +200
- +500
- +1000

可以连续点击累加，例如：
- 点击 +500 → 显示 ¥500
- 再点击 +200 → 显示 ¥700

### 3. **语音输入（如果浏览器支持）**
- 点击麦克风图标 🎤
- 说出金额（如"五百"）
- 系统自动识别并填入

**支持的浏览器：**
- Chrome/Edge（推荐）
- Safari（iOS/macOS）
- 其他现代浏览器

### 4. **智能分类**
- 支出自动归类为"其他支出"
- 收入自动归类为"其他收入"
- 如果分类不存在，自动创建

### 5. **事后补充**
快速记账后，可以在"查询统计"页面：
- 编辑记录
- 补充人物关系
- 修改分类
- 添加详细备注

## 📱 使用方法

### 方式一：从首页进入
1. 打开应用首页
2. 点击"⚡ 快速记账"卡片
3. 输入金额（或点击快捷按钮）
4. 点击"保存"

### 方式二：直接操作
```
步骤1: 输入金额 800
步骤2: （可选）输入姓名 "张三"
步骤3: （可选）选择类型 收入/支出
步骤4: 点击保存
```

**就是这么简单！**

## 🎯 与"记一笔"的区别

| 特性 | 快速记账 | 记一笔（完整流程） |
|------|---------|------------------|
| 必填项 | 仅金额 | 金额+姓名+分类+日期 |
| 人物关系 | 可选 | 可选择详细关系 |
| 分类 | 自动默认 | 手动选择 |
| 日期 | 自动今天 | 可选择任意日期 |
| 速度 | ⚡⚡⚡ 极快 | 🐢 较慢但详细 |
| 适用场景 | 日常快速记录 | 重要场合详细记录 |

## 🔧 技术实现

### 数据结构
快速记账创建的记录包含：
```javascript
{
  type: 'expense',           // 类型：支出/收入
  amount: 800,               // 金额（必填）
  name: '张三',              // 姓名（可选，默认"未命名"）
  relationshipId: null,      // 关系ID（无）
  categoryId: 1,             // 分类ID（自动选择默认）
  date: '2024-01-15',        // 日期（自动今天）
  note: '婚礼'               // 备注（可选）
}
```

### 语音识别
使用 Web Speech API：
- 语言：中文（zh-CN）
- 模式：单次识别
- 提取：数字部分
- 容错：未识别到数字时提示

### 自动备份
每次快速记账后会自动创建轻量备份，确保数据安全。

## ❓ 常见问题

### Q1: 快速记账会不会丢失数据？
**A:** 不会。快速记账和完整记账使用相同的数据库，数据安全性完全一致。

### Q2: 没有选分类怎么办？
**A:** 系统会自动使用默认分类（"其他支出"或"其他收入"）。你可以在查询页面后期修改。

### Q3: 语音识别不准确怎么办？
**A:** 
- 在安静环境下使用
- 清晰地说出数字
- 也可以手动输入，语音只是辅助

### Q4: 能否设置默认金额？
**A:** 目前不支持，但可以连续点击快捷按钮累加。这个功能可以考虑加入未来更新。

### Q5: 快速记账后如何补充详细信息？
**A:** 
1. 进入"查询统计"页面
2. 找到刚才的记录
3. 点击编辑按钮
4. 补充人物关系、修改分类等
