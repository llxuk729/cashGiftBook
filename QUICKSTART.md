# 快速启动指南

## 🎉 项目已初始化完成！

你的礼金记账应用已经创建完成，开发服务器正在运行中。

## 📱 立即体验

### 访问应用
打开浏览器访问：**http://localhost:5173/**

### 主要功能
1. **首页** - 查看统计概览和最近记录
2. **记一笔** - 添加新的礼金记录
3. **人物关系** - 管理家族关系树
4. **查询统计** - 多维度查询和统计
5. **数据备份** - 导出/导入备份文件
6. **设置** - 版本管理和数据清理

## 🎯 使用流程

### 第一步：建立人物关系
1. 点击底部"关系"标签
2. 从"我"开始，点击右上角"+"添加家人
3. 选择关系（父亲、母亲、兄弟姐妹等）
4. 可以逐层添加，构建完整的家族树

### 第二步：记录礼金
1. 点击底部"记账"标签或首页"记一笔"
2. 选择类型（收入/支出）
3. 输入金额
4. 选择人物（从关系树中选择）
5. 选择分类（结婚、周岁、考大学等）
6. 选择日期
7. 添加备注（可选）
8. 点击保存

### 第三步：查询统计
1. 点击底部"查询"标签
2. 可以按人物、分类、时间筛选
3. 查看实时统计数据
4. 浏览详细记录列表

### 第四步：定期备份
1. 点击首页右上角"备份"按钮
2. 或进入"设置" -> "数据备份"
3. 点击"导出备份文件"
4. 将 JSON 文件保存到安全位置

## 💡 小贴士

### 数据安全
- ⚠️ 数据存储在浏览器本地
- ⚠️ 清除浏览器缓存会丢失数据
- ✅ 每周系统会提醒备份
- ✅ 建议将备份文件发送到微信收藏或网盘

### 人物关系技巧
- 从"我"开始逐层添加
- 可以使用"自定义"关系添加特殊关系
- 关系路径会自动生成（如：我 > 父亲 > 叔叔）
- 删除人物前需先删除其子节点

### 查询技巧
- 可以组合多个筛选条件
- 点击清除图标快速重置筛选
- 统计数据会随筛选实时更新

## 🐛 遇到问题？

### 常见问题

**Q: 页面显示空白？**
A: 检查浏览器控制台是否有错误，确保开发服务器正常运行

**Q: 数据丢失了？**
A: 检查是否清除了浏览器缓存，可以从备份文件恢复

**Q: 如何在手机上访问？**
A: 
1. 确保手机和电脑在同一 WiFi
2. 运行 `npm run dev -- --host`
3. 用手机浏览器访问显示的局域网地址

**Q: 如何部署到服务器？**
A: 
1. 运行 `npm run build` 构建
2. 将 `dist` 目录上传到 Web 服务器
3. 或使用 Vercel、Netlify 等免费托管服务
